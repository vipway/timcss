# TimCSS React Mobile Example

This example shows the platform-neutral mobile atomic utilities in a React-style app surface.

## What it demonstrates

- `px-page` page gutters
- `py-section` section spacing
- `p-card` card padding
- `gap-section` and `gap-card`
- `h-control` and `min-h-touch`
- `rounded-card` and `rounded-control`
- `bg-primary`, `bg-surface`, `text-on-primary`
- `pressed:` and `disabled:` state variants

## Commands

From the repository root:

```bash
pnpm run timcss:example:react:build
pnpm run timcss:example:react:inspect
pnpm run timcss:example:react:doctor
```
